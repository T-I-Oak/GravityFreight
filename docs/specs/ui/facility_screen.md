# UI仕様: 施設画面

## 目的
- セクター成功後に発生する拠点イベント（交易所/整備工場/闇市場）を提供する。

## 想定する状態（シーン）
- FACILITY

## 施設種別
- TRADING_POST
- REPAIR_DOCK
- BLACK_MARKET

## 主要UI
- 施設名/説明
- 所持金/所持品
- 取引/整備/購入の操作UI
- 次セクターへ進む導線

## 入力
- マウス/タップ操作

## 遷移
- FACILITY → PLAY（次セクター開始 / BUILDING）
- FACILITY → GAME_RESULT（次セクター開始条件を満たせずゲームオーバーとなる場合）

## 表示スタイル
- 原則: マットスタイル
- 闇市場など例外が必要なら後で定義
