# Specification: Item Class (Skeleton)

## 1. 役割と責務 (Role & Responsibility)

- **所属ドメイン**: Entity Domain
- **生存期間**: Game Lifecycle（インスタンスとして）
- **役割**: ゲーム内の全アイテムの基底。
- **責務**:
  - 個別の属性（ID、現在耐久値、強化状態など）の保持。
  - レアリティや耐久度から算出される、自身の基準価格（査定価値）の提供。
  - ※具体的なプロパティ・メソッドは、シナリオ策定プロセスの中で必要になったものだけを後から追記する。
