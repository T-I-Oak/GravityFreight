# Gravity Freight V2: Backlog Archive

## Version 1.6.1 (2026/04/20) - タイトル画面の再設計とレイヤー構造の最適化
- 新しい UI アーキテクチャに基づき、タイトル画面を再定義。レスポンシブなロゴ、カプセル型ボタン、背景演出による没入感を向上。
- Z-Index 階層の標準化を行い、HUD やパネルの重なり順の不整合をトークンベースで解決。
- インタラクションレイヤー（Neon）をクリーンアップし、ボタンのホバー・アクティブ時のフィードバックを精密に調整。

## Version 1.6.0 (2026/04/18) - UI基盤の刷新とアイテムカードのデザイン再現
- アイテムカードのビジュアルを V1 デザイン（平行四辺形状の耐久度ゲージ、高密度な数値バッジ、タイトルの自動省略等）へ完全復元し、視認性と情報密度を向上。
- マット・ネオン・印刷の複数スタイル切り替えに対応可能な 3層 CSS アーキテクチャを導入。
