# GRAVITY FREIGHT: ORBITAL ECHOES - 要求仕様書 (アイテムカタログ)

---

## 1. 構成アイテムの分類と役割
本作に登場する「アイテム」は、その具体的データ構造と役割に応じて以下のように分類されます。

### 1.1 ロケット構成パーツ (Components)
ロケットの属性（重量、スロット、精度、取得範囲）を決定する、恒久的なデータ・アイテムです。
- Chassis / Logic / Module: それぞれがロケットの重量や精度に影響します。特にモジュールは、同じ種類のパーツを複数搭載した場合でも、それぞれが個別の耐久度や強化状態を持つ独立した実体として管理されます。

### 1.2 出撃装備 (Launch Gears)
フライトという一時的なセッションにのみ適用・消費されるアイテムです。
- Launcher: 出撃における「パワー（初速）」と「精度倍率」、および「耐久度（Charges）」を定義。
- Booster: 発射時に消費され、1フライトの間だけ特定のボーナスを付与する消耗データ。

### 1.3 収集アイテムの挙動 (Collected Items)
宇宙空間（重力源の周囲）には、すべてのカテゴリー（パーツ、装備、リソース）のアイテムがランダムに配置される可能性があります。
- 初期配置規則: セクター生成時に、母星（Home Star）を除くすべての重力源（星）の周囲に対し、ランダムなアイテムが 1〜2 個配置されます。
- 取得と保持: 宇宙空間で取得されたアイテムは、カテゴリーを問わず即座にロケットの「保持状態」となります。この状態のアイテムはフライトが終了（成功・帰還・失敗）するまで、インベントリや所持金には反映されません。

## 2. 性能算定（スペック計算）の規則
ロケットの最終的なフライトスペックは、構成する全アイテムの値を以下の規則に従って集計・正規化します。

| 集計方式 | 適用項目 | 計算方法 |
| :--- | :--- | :--- |
| 加算 (Additive) | 重量, スロット数, 基本予測精度, 基本取得範囲 | `Total = Σ(Value)` |
| 乗算 (Multiplicative) | 予測倍率, 取得倍率, 重力倍率, アーク倍率 | `Total = 1.0 * Π(Multiplier)` |
