## Version 1.0.0 に向けたロードマップ

### メタゲーム (Meta-game)
- [ ] **ランキング機能の実装**:
    - **Sector ランキング**: 到達した最大セクターの記録。
    - **Score ランキング**: 累計スコアの記録。
    - **Collected Items ランキング**: 累計貨物収集数の記録。
        - インベントリにアイテムを追加する処理を共通処理として実装し、その処理を呼び出したときにカウントするようにする
        - 初期インベントリはカウントの対象にしない
        - コインやcargoはインベントリに入るものではないが、獲得したらカウントするようにする
    - ランキングはlocalStorageに保存する
    - ランキングは20位まで保存する

### ゲームシステム (Game System)
- [ ] 「ラスボス」のような大型ターゲットを追加
    - 今のルールだけでは同じステージの繰り返しになってしまうので、5セクターごとなどに何か目標になるものを追加したい
    - 例: star itemsとして回収必須のアイテムを複数個所に追加し、すべて回収するまでアークがdisabledになる（lost扱い）
- [ ] [実験] 重力がマイナスの星を作ったらどうなるか
    - 1個だったらどうなるか
    - すべてだったらどうなるか
    - 新しい体験になりそうなら、「ラスボス」の代わりに追加をしたい
- [ ] ストーリーの導入
    - cargoボーナスを獲得したときにストーリーが進む仕組みを導入する
    - ストーリーはゲームごとに最初の分岐から始まる
    - 出口（3種類）に応じてストーリーが分岐する
        - 1回目で3分岐、2回目で9分岐、3回目で27分岐（ストーリーはここまで）
    - deliveredの報酬アイテムとして仮想的に「手紙」を追加する（フラグを立てるだけ）
    - ゲーム中のスコアー表示のあたりにメールアイコンを追加し、ホバー（タップ）したらメールが読めるようにする
    - メールアイコンは出口に応じたカラーで表示される
    - 未読のメールには未読マークを付ける
    - local storageに、どのメールを取得したかの記録を残す
        - 1回読んだメールは未読マークを付けない

### ゲームオーバー画面
- [ ] **ゲームオーバー画面機能追加**: ランキング表示の追加、総合評価（グレード）の表示追加
    - Delivered の結果表示をなくし、Collected Items の結果表示に変更
    - Sector, Collected, Score の各項目について、対応するランキングとグレードを表示する
        - ランキングとグレードは、点数表示よりもフォントサイズを落として右寄せよりやや中央寄りにインデントして表示する
        - ランキングは点数表示の次の行、グレードはランキングの次の行に表示する
        - ランキング21位以降は圏外として表示する
        - ランキングベスト3は少しだけ目立つようにする（色を変えるなど）
        - グレードは SS, S, A, B, C, D, E の7段階
        - グレードによって色を変えるが、レシート風の雰囲気を壊さないこと
    - 総合ランキングも SS, S, A, B, C, D, E の7段階
    - 総合ランキングは、レシートにスタンプを押したようにアニメーションで表示する
        - cssで斜めに傾けて、かすれた雰囲気を出す
        - グレードが高いほど大きなスタンプにする
        - Scoreの表示の下にウォーターマークのスペースがあるので、そこに表示する

## Version 3.0.0 に向けたロードマップ

### UI・演出の改善 (UX/UI)
- [ ] **アイテムごとの画像設定**: カード背景に各アイテムの個別画像を表示し、視認性と没入感を向上。
