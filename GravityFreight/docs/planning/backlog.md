## Version 3.0.0 に向けたロードマップ

### UI・演出の改善 (UX/UI)
- [ ] **アイテムごとの画像設定**: カード背景に各アイテムの個別画像を表示し、視認性と没入感を向上。
- [ ] **Unit画像の自動生成**: 選択された Chassis、Logic、Module を合成し、建造されたロケット固有の画像を自動生成してカード背景に表示。
- [ ] **Unitに応じたロケット描画**: 現在の汎用形状（青い三角）ではなく、選択された Unit の素材や形状を反映したリッチなロケットモデルが飛行するように拡張。
- [ ] **モジュール発動時の視覚効果（FX）の刷新**: 「スター・ブレイカー」の破壊粒子、クッションの衝撃波、スラスターの火花など、主要モジュールの発動エフェクトを一貫性のあるプレミアムな品質で実装。
