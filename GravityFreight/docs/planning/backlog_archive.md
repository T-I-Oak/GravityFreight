## Version 0.7.0 (2026/04/02) - 仕様整合性と経済システムの安定化

### 仕様整合性の確保
- [x] **目的地報酬および配送ボーナスの数値検証とロジック修正**: 基礎報酬 (+2000~5000) とボーナス (+1500) の合算を検証・修正。
- [x] **出現率とセクターしきい値の重み付け抽選の検証**: `14 + S` の計算式とレアリティごとの重み付けが仕様通りであることを検証。
- [x] **複数保険モジュールの重複支払い検証**: 保険モジュールの装備数に応じた保険金支払いロジックの正当性を検証。

### 貨物・経済・バランス調整 (Economy & Balance)
- [x] **デバッグコードの削除と配置数の正常化**: `MissionSystem.js` 内のアイテム強制注入コードを削除し、星ごとのアイテム配置数（1〜2個）の制約を遵守。
- [x] **ゴール通過報酬の正常化**: 貨物を所持していない場合でも、ゴール到達時にボーナスアイテムが付与されるように改善。
- [x] **アイテムデータの日本語化と仕様書整合**: `Data.js` の貨物アイテム名を日本語に統一し、`spec.md` の説明文と完全一致するように修正。

## Version 0.9.0 (2026/04/02) - アイテムバランス刷新と保険システムの拡張

### バグ修正と品質向上 (Bug Fixes & Quality)

- [x] **経済システムおよびアイテムデータの整合性検証**: 経済システムおよびアイテムデータの整合性を検証。

## Version 0.9.2 (2026/04/03) - UX 高速化と開始演出の導入

### UI・演出の改善 (UX/UI)
- [x] **起動パフォーマンスの最適化 (Lag Elimination)**: 
    - タイトル画面からビルド画面への遷移時に設定されていた 1秒間の固定遅延 (`setTimeout`) を削除。

    - アイテム獲得なし時の文言を「回収アイテムなし」に固定し、点滅アニメーションを削除。

### パフォーマンスと品質 (Performance & Quality)

- [x] **バージョン管理とキャッシュ対策の徹底**: 
    - `Game.js`, `index.html`, `package.json` のバージョンを `0.9.2` に一括更新。
    - `index.html` の外部スクリプト読み込みに `?v=0.9.2` 形式のクエリパラメータを付与し、キャッシュによる不整合を防止。
- [x] **デグレード防止の確認**: 既存の整合性を確認し、バージョン更新に伴う期待値の同期も完了。

## Version 0.10.0 (2026/04/03) - モバイル入力対応

### 入力・操作性 (Mobile Controls)
- [x] [重要・BUG] iPad / Android で操作できなくなっている
    - `pointer/touch` 経由で入力を受け取り、iOS/Android でもエイミング・回転・パン操作が反映されるように修正
    - エイミング中でも 2本指操作を優先し、2本指ドラッグはパン（`cameraOffset`）、2本指ピンチはズーム（`zoom`）として動作するよう調整

### 物理挙動の整合 (Physics Consistency)
- [x] [重要・BUG] 予測線と実際の軌道がずれている（予測線では crash していないのに crash した）
    - `aiming` 開始時にロケットの総質量 (`rocket.mass`) を `ship.mass` に同期し、予測線と実フライト双方が同一の質量パラメータでシミュレーションされるように修正
    - `Game.test.js` に質量同期テストを追加し、`aiming` 開始後に `ship.mass === rocket.mass` であることを保証

### Cargo / リザルト表示 (Cargo & Result)
- [x] [BUG] 母星にアイテムがあるとき、発射したロケットが母星のアイテムを回収する仕組みが機能していない
    - `PhysicsOrchestrator.handleFlight` で母星が回収対象から除外されていたため、母星上のアイテムが回収されなかった問題を修正
- [x] [BUG] リザルト画面で、正しくcargoを配達したときにもらえるカーゴボーナスが表示されない（もらえているのかも不明）
    - 配送成功時ボーナスの表示名を `Cargo Bonus` に統一し、リザルトのボーナス一覧に確実に表示されるよう修正。
