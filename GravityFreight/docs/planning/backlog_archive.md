# Gravity Freight 開発バックログ アーカイブ

このファイルはバージョンごとの改版履歴です。
ユーザー体験（追加機能、UIなどの改善、影響のある不具合修正）のみを記載し、内部作業のログは含みません。

## Version 0.34 (2026/04/12) - 用語の標準化とシステム基盤の強化

### 機能改善 (Enhancement)
- ズーム率の永続化: マップのズーム倍率を自動保存し、次回起動時やリロード時にも前回の設定を維持するように改善。
- 用語定義の標準化: ゲーム全体における「機体」という呼称を「ロケット」「シャーシ」「ロケット構成パーツ」へと整理・統一。
- ストーリーデータの最新化: 全 39 エピソードのストーリー定義を最新の内容に更新。

### 不具合修正 (Bugfix)
- インパクト・クッションの挙動修正: インパクト・クッション搭載機が母星（Home Star）に衝突した際、バウンドせずに正常に帰還成功判定となるよう修正。

## Version 0.33 (2026/04/11) - チュートリアル・エクスペリエンスの刷新


### 機能追加 (Feature)
- プレミアムチュートリアルシステム: 7枚の動的スライド、高品質な図解アセット、および視認性と操作性を重視した専用 UI デザインを採用した新しい「How to Play」システムを実装。

## Version 0.32 (2026/04/09) - システム安定性の向上と重要不具合の解消

### 機能改善 (Enhancement)
- 操作性の向上: 非エイミング状態およびエイミング中において、キー（Shift/Ctrl）操作を伴うパニングが正常に動作しない不具合を修正。
- オーディオ操作性の向上: 音量設定を 0% にした際、設定画面でのプレビュー音も含めて完全に消音されるよう仕様を厳格化。

### 不具合修正 (Bugfix)
- 軌道予測の不一致解消: 閃光推進剤（Flash Propellant）使用時、予測線と実際の飛行ルートが乖離してしまう不具合を修正。
- アイテム回収の修正: 重力偏向ロジック等の特定パーツ装備時にアイテムを収集できなくなる不具合を修正。

## Version 0.31 (2026/04/09) - システムロジックの適正化と表示レイアウトの洗練

### 機能追加 (Feature)
- 最新ミッションのハイライト: ランキング画面において、全カテゴリーを通じて「最も新しいプレイ結果」を視覚的に強調する表示機能を追加.
- 0値ランキングのサポート: スコア等が 0 の記録であっても、正当な実績としてランキングに反映し比較できるように拡張.

### 機能改善 (Enhancement)
- 経済システムロジックの適正化: レアリティごとのベース価格を定数管理し、未定義データへのデフォルト処理を廃止してデータ整合性を強化.
- 耐久度表示とUIレイアウトの最適化: ロケット内での合算表示と単体カードでの個別表示の切り分けを実装し、情報のまとまりを重視した配置へ微調整.

### 不具合修正 (Bugfix)
- 打ち上げ時の耐久度全快バグ: ロケット打ち上げ時に、搭載モジュールの現在耐久値が最大値にリセットされてしまう不具合を修正.
- ランキング順位計算の不具合: 0値や同点時の優先順位判定、および20位以下のデータ整理ロジックの不備を解消.
- 解体時のパーツ状態の維持: ロケット解体時に、パーツの耐久度や強化状態がリセットされず、搭載時の状態を保持したままインベントリへ戻却されるよう不具合を修正.

## Version 0.30 (2026/04/09) - 統合ランキングシステムの導入とUIの標準化

## Version 0.29 (2026/04/08) - 斥力星（Repulsive Star）と ANOMALY レアリティの導入

### 機能追加 (Feature)
- 斥力星（Repulsive Star）システム: 初のマイナス重力（反発力）を発生させる星の概念を導入。
- 新レアリティ「ANOMALY」: 既存を凌駕する最上位ランクを新設。該当アイテムを保持する星は斥力を持つ仕様を追加。
- 周期的な環境干渉: 5セクターごとに環境の引力と斥力が逆転する特別ルールを追加。

### 機能改善 (Enhancement)
- 斥力星の視覚表現: 危険を識別できるように斥力星を専用カラー（マゼンタ）で描画するように変更。
