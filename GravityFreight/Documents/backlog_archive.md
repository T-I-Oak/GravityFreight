## Version 0.7.0 (2026/04/02) - 仕様整合性と経済システムの安定化

### 仕様とテストの整合性確保 (Specs & Tests)
- [x] **目的地報酬および配送ボーナスの数値検証とロジック修正**: `07_collection_economy.test.js` を作成し、基礎報酬 (+2000~5000) とボーナス (+1500) の合算を検証・修正。
- [x] **出現率とセクターしきい値の重み付け抽選の検証**: `14 + S` の計算式とレアリティごとの重み付けが仕様通りであることをテストで証明。
- [x] **複数保険モジュールの重複支払い検証**: 保険モジュールの装備数に応じた保険金支払いロジックの正当性を検証。

### 貨物・経済・バランス調整 (Economy & Balance)
- [x] **デバッグコードの削除と配置数の正常化**: `MissionSystem.js` 内のアイテム強制注入コードを削除し、星ごとのアイテム配置数（1〜2個）の制約を遵守。
- [x] **ゴール通過報酬の正常化**: 貨物を所持していない場合でも、ゴール到達時にボーナスアイテムが付与されるように改善。
- [x] **アイテムデータの日本語化と仕様書整合**: `Data.js` の貨物アイテム名を日本語に統一し、`spec.md` の説明文と完全一致するように修正。

## Version 0.9.0 (2026/04/02) - アイテムバランス刷新と保険システムの拡張

### アイテム・バランス調整 (Items & Balance)
- [x] **発射台およびブースターの名称刷新と性能調整**:
    - 発射台を `LN1200-N` / `PR1000-N` 系列へ型番化し、直感的な性能把握を可能に。
    - 「アーク・エクスパンダー」への名称変更と、拡大倍率の強化（1.2倍 → 2.0倍）。
- [x] **ロスト保険の適用範囲拡大**:

    - 貨物を所持していない機体消失時でもパーツ査定額を支払うように修正。

### バグ修正と品質向上 (Bug Fixes & Quality)
- [x] **リザルト画面のゴールボーナス累積バグの修正**: `Game.reset()` 時のデータ初期化漏れを解消。
- [x] **自動テストによる品質保証**: 経済システムおよびアイテムデータの整合性を検証するテストスイートの拡充。
