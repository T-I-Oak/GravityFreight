# Gravity Freight 開発バックログ (v1.0.0に向けたロードマップ)

初版リリースに向けて必要な機能をカテゴリ別に整理しました。

## 0. 重要バグ修正 (Critical Bug Fixes)
- [x] **衝突判定の突き抜けバグの修正 (v0.4.1)**: 高速移動時に星を通り抜けてしまう現象（物理ステップを飛び越える、またはCCDの判定漏れ）の再検証とCCD（連続衝突判定）の強化。
- [x] **アセンブリ時のスロット数超過バグ (v0.4.1)**: モジュール選択後にスロット数の少ないシャーシへ変更した際、制限を超えて搭載できてしまう問題の修正。
- [x] **ブースターの不正消費バグ (v0.4.2)**: 燃料を選択した状態で機体を建造すると、発射前にもかかわらず燃料が消費されてしまう問題の修正。
- [ ] **【不具合疑い】精密発射台のパラメータ**: 「予測線強化」と記載されているが、現在のパラメータ設定が意図した通りの効果を発揮しているか再検証。

## 1. UI・演出の改善 (UX/UI)
- [x] **ビルドパネルの空状態表示 (v0.4.2)**: すべてのカテゴリー（シャーシ、ロジック、拡張、ブースター、自機一覧）において、在庫がない場合にプレースホルダーを表示する機能。
- [ ] **リザルト画面の強化**: 終了時（CLEAR/CRASH）に取得したアイテムと獲得スコアの内訳を表示。
- [x] **Mission HUD の導入 (v0.4.2)**: タイトル、セクター、スコアを統合した視認性の高いパネルを左上に配置。
- [x] **パネルの折りたたみ・透過演出 (v0.4.2)**: ガラスモーフィズムを採用し、不要なパネルを上方に折りたたむ機能の実装。
- [x] **スコア演出の改良 (v0.4.2)**: ゴール時に滑らかにカウントアップするアニメーション演出。
- [x] **帰還時アイテムの即時インベントリ化 (v0.4.1)**: 帰還（母星命中）した際、取得したアイテムを一時預かりではなく即座にインベントリに加算する。
- [ ] **新ブースター「高出力ブレード」**: 発射時のパワー（初速）を底上げする効果を持つブースター。

## 2. 貨物・経済システムの導入 (Cargo & Economy)
- [ ] **貨物の一般出現化**: Cargoアイテムを `COMMON` として星にランダム配置。
- [ ] **配送報酬ロジック**:
    - 対応する色のゴールへ届けると追加報酬（コイン等）を付与。
    - 報酬内容はゴールするまで不明（難易度の高いゴールほど高報酬）。
- [ ] **コイン（通貨）システム**: アイテムとして取得、または配送報酬として獲得。
- [ ] **新モジュール「機体保険」**: CRASH/LOST時に、機体価値に応じたコインを受け取れる効果。

## 3. ステージイベント・ショップ (Stage Events)
- [ ] **出口ごとの特殊イベント**:
    - **HARD (DANGER)**: 到達後、次ステージから星の生成数が永続的に +1。
    - **NORMAL**: 「整備工場」イベント。Unitの解体（インベントリ還元）やコインによる燃料（Charges）補充。
    - **SAFETY (SAFE)**: 「ショップ」イベント。コインによるアイテムの売買。

## 4. ランキング・リプレイシステム (Social & Replay)
- [ ] **ランキング(1) 通算スコア**: ゲームオーバー時に到達ステージとポイントを記録・表示。
- [ ] **ランキング(2) ステージ別**: ステージごとの獲得ポイントを記録。
- [ ] **リプレイ機能**: ランキングされたプレイを再現。
    - ※リプレイの完全再現のため、飛行中のユーザー介入を制限（初期状態で弾道が確定する仕様）。

## 5. 基盤・その他の拡張 (Infrastructure & Platform)
- [x] **タブレット対応 (v0.4.0)**: iPad/Android等のモバイルブラウザでの操作最適化。
- [ ] **オーディオ実装**: BGMおよび効果音（SE）の追加。
- [ ] **アチーブメント**: 実績システムの導入。

## 6. エフェクト・視覚効果の強化 (Visual Effects)
- [ ] **星の破壊エフェクト**: 「Star Breaker」による星の破壊時、粒子が飛び散るような視覚効果を追加。
- [x] **背景の小さな星にゆらぎやアニメーション (v0.4.2)**: 3Dスターフィールド（中央から外側への移動）と明滅効果によるダイナミックな演出。
- [x] **マップ・カメラ操作の追加 (v0.4.2)**: パララックス効果を伴うパン操作の実装。
- [ ] **背景のワープ演出**: ステージクリア（セクター移動）時、背景の星が加速し、現在のステージが手前に拡大消去、次のステージが奥から拡大出現する演出。
- [ ] **状況に応じた動的エフェクト**: 加速時やアイテム取得時、および境界付近での警告演出などの強化。

## 7. v2 以降に向けた機能要望 (Future Roadmap)
- [ ] **アイテムごとの画像設定**: カード背景に各アイテムの個別画像を表示し、視認性と没入感を向上。
- [ ] **Unit画像の自動生成**: 選択された Chassis、Logic、Module を合成し、建造された機体固有の画像を自動生成してカード背景に表示。
- [ ] **Unitに応じたロケット描画**: 現在の汎用形状（青い三角）ではなく、選択された Unit の素材や形状を反映したリッチなロケットモデルが飛行するように拡張。
