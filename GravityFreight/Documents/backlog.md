# Gravity Freight 開発バックログ (v1.0.0に向けたロードマップ)

初版リリースに向けて必要な機能をカテゴリ別に整理しました。
完了した項目は [backlog_archive.md](./backlog_archive.md) に移行されました。

## 重要バグ修正 (Critical Bug Fixes)
- [x] **パネルスクロール時のエイムずれ**: パネルをドラッグしてスクロールした際、意図せずエイム（発射角度）が動いてしまう問題の修正。 (2026-03-27完了)
- [ ] **不一致の荷物を持ってゴールしたとき、リザルト画面で「Cargo (unmatched)」と「Cargo Reword」で重複した値が表示される**
- [ ] **cargo bonusでcoinを取得してもサマリーにその金額が表示されない**
- [x] **[BUG] 同種の通常アイテムとエンハンスアイテムを同時所持している際、エンハンスアイテムを選択しても通常アイテムが優先して選択されてしまう** (2026-03-27検証完了)
- [ ] **発射準備時の向きの不一致**: ドラッグでのエイム時、船の座標は更新されるが向き（rotation）が同期されず、発射時に意図しない向きになる。（※再現方法が判明した時点で着手）
    - ドラッグ中、機体の位置は正しく移動するが、向き（回転）だけが同期せず固定されてしまう。
- [ ] **セクター1でのレアアイテム出現不具合**: セクター1で「スター・ブレイカー」等のレアアイテムが抽選される。低難易度セクターでのドロップバランス不備の修正。
- [x] **ユニットカードの表示不具合**: Unitのカードに、搭載されているModuleの情報が重複して表示されてしまう問題の修正。 (2026-03-27完了)

## UI・演出の改善 (UX/UI)
- [ ] **星のアイテム表示の最適化**: 1つの星に大量のアイテム（Cargo/Parts）がある場合、ホバー表示が画面外にはみ出す問題の解消。
    - アイテムリストのグリッド表示化、または「...他○件」といったサマリー表示の検討。
- [ ] **アイテムカードの説明文の見直し**: 現在のゲームシステムや表現に合わせて、各アイテムの説明文（フレーバーテキスト）の内容を全面的に再確認・修正。
- [x] **マップを回転可能にする**: パネルでマップが隠れる際の視認性向上。回転時、ゴールラベルの向きが不自然にならないよう配慮する。 (2026-03-28完了)
- [x] **Unit カードの耐久度マーク表示の統一**: Unitのカードに表示されるModuleの耐久度マークのスタイルを、通常のカードに表示される耐久度マークと同じスタイルに統一する (2026-03-27完了)

## ステージイベント・ショップ (Stage Events)
- [ ] **出口ごとの特殊イベント**:
    - **HARD (DANGER)**: 多額のコインを支払って、ランダムに抽選されたレアリティの高いパーツを入手できる。到達後は、次ステージから星の生成数が永続的に +1。
- [ ] イベント時のzoom抑止: イベント画面でスクロールすると、マップに戻ったときに拡大率が変わってしまう問題を修正
- [ ] **イベント画面のコイン増減演出の統一 (v0.5.4)**: リザルト画面の演出と統一。余計なポップアップが残っているので消す。
- [ ] **イベント画面のレイアウト刷新**:
    - [ ] 現状のスクロール必須なレイアウトを廃止し、基本構成をスクロールなしで完結させる（視認性と操作性の向上）。
    - [ ] イベント中にインベントリを確認できる仕組みを追加する
        - REPAIR DOCK でLauncherを回復する必要があるのか、インベントリに十分Launcherがあるのかが分からない

## 貨物・経済・バランス調整 (Cargo, Economy & Balance)
- [x] **Repair Dock の解体ボーナス仕様改善 (v0.5.5)**: 属性を持たないパーツにもボーナスがつくように `enhanceItem` を修正
- [x] **Data.js から spec.md へのデータ反映 (v0.5.5)**: 大幅更新された `Data.js` の内容を `spec.md` のアイテム管理表に反映
- [ ] **結果画面のスコア/コイン重複バグ修正**: リザルト画面でスコアやコインが二重に加算される問題を調査・修正
- [ ] **発射時の向きズレ修正**: 発射ボタンを押した瞬間にわずかに向きがズレる問題を修正
    - 変更後: イベント（Repair Dock / Black Market 等）での **消費コイン減額**。1つにつき `-10%` (重複可、最大 `-50%` )。
- [ ] ロスト保険のバランス調整: Lostしたときだけでなく、Crashしたときにも保険金が受け取れるように変更。
- [ ] 高反応燃料（高反応燃料パック）のバランス調整: powerMultiplierを1.2に変更。

## エフェクト・視覚効果の強化 (Visual Effects)
- [ ] **星の破壊エフェクト**: 「Star Breaker」による星の破壊時、粒子が飛び散るような視覚効果を追加。
- [ ] **背景のワープ演出**: ステージクリア（セクター移動）時、背景の星が加速し、現在のステージが手前に拡大消去、次のステージが奥から拡大出現する演出。
- [ ] **状況に応じた動的エフェクト**: 加速時やアイテム取得時、および境界付近での警告演出などの強化。

## v2 以降に向けた機能要望 (Future Roadmap)
- [ ] **アイテムごとの画像設定**: カード背景に各アイテムの個別画像を表示し、視認性と没入感を向上。
- [ ] **Unit画像の自動生成**: 選択された Chassis、Logic、Module を合成し、建造された機体固有の画像を自動生成してカード背景に表示。
- [ ] **Unitに応じたロケット描画**: 現在の汎用形状（青い三角）ではなく、選択された Unit の素材や形状を反映したリッチなロケットモデルが飛行するように拡張。
