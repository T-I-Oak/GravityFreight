# 開発の完了 (Finish Development)
このワークフローは、開発した内容を検証し、プロジェクトドキュメントを整理して master ブランチへ統合する手順を自動化します。

1. **一時ファイルの削除とバージョン更新**
   - ファイルやディレクトリ名に `tmp_` がつく一時ファイルが存在する場合は、開発完了の処理前にすべて削除する。
   - `package.json` のバージョンを更新する。これのみで JS/HTML への反映とキャッシュ問題の解決が Vite により自動で行われる。

2. **仕様の最新化**

3. **[STOP!] 最終承認**
   - 実施した内容を報告し、ユーザーの最終承認を待機する。

4. **統合 (Merge & Push) と 自動デプロイ**
   - 承認後、`master` ブランチへマージし、リモートへプッシュを行う。
   - **プッシュにより GitHub Actions が発火し、GitHub Pages へのデプロイが自動実行される。**
